An interactive installation inspired by John Cage’s method for composing with chance.

Related project: Dublab
Collaborator: Ali Torbati
Category: Interactive
Year: 2018

Design as Performance

My proposed identity for Dublab is based on a set of unique tools for graphic creation inspired by music.

This interactive installation allows guests to generate their own posters and output them in real time via a hacked dot matrix printer, honoring the time-based nature of musical performance.
