An interactive installation inspired by John Cage’s method for composing with chance that gives users the power to create and print their own posters live.

Related project: Dublab
Collaborator: Ali Torbati
Category: Interactive
Year: 2018

Design as Performance

I created a proposed identity for Dublab based on a unique set of tools for graphic creation inspired by sound and music.

This interactive installation allows guests to generate and output their own posters in real time via a hacked dot matrix printer. Rather than a singular event poster, a controlled set of parameters allows for infinite graphic variations with a user’s personal message.
